# Human Emotion and Stress Detection

## Overview
This project detects human emotions and stress levels using Deep Learning and Computer Vision techniques. It works in real-time using webcam input.

## Features
- Real-time face detection
- Emotion classification
- Stress analysis
- Audio alert system

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- NumPy

## How to Run
1. Install dependencies
2. Run the notebook or main script
